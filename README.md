## About
브레인 스토밍에서 나온 아이디어를 마인드맵으로 구성할 수 있는 실시간 아이디에이션 기획 툴입니다.

## Feature
- 아이디어를 카드 형식으로 빠르게 작성할 수 있다.
- 마인드맵에 아이디어 카드를 추가하고 연결할 수 있다.
- 프로젝트를 개인과 팀으로 구분해서 생성할 수 있다.
- 팀프로젝트는 실시간으로 마인드맵 컨텐츠를 공유할 수 있다.
